матеріали для сайту про баскетбол


Цей репозиторій містить матеріали для створення інформаційного сайту про баскетбол. 
Тут зібрані статті, нотатки та медіафайли, які можна використати для розробки контенту.

## Структура проєкту
- `article.md` — основна стаття про баскетбол
- `drafts/` — чорнові нотатки та матеріали
- `images/` — зображення та графіка для сайту

## Призначення
Репозиторій створений для навчального проєкту та демонстрації структури контентного сайту.
